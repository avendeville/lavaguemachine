Le système vendevillien révèle les mécaniques invisibles qui structurent les institutions et propose une architecture de lucidité pour les réparer.

Les trois portes d’entrée

Lire les mécaniques — comprendre ce qui agit réellement dans les institutions.
Stabiliser le langage — déplacer les catégories, clarifier les formes, neutraliser les dérives.
Réparer l’institution — restaurer la cohérence, reconstruire les organes, stabiliser les systèmes humains.

Cosmologie vendevillienne

Le système vendevillien lit les institutions à travers leurs mécaniques, leurs récits et leurs organes. Il agit non par la psychologie, mais par la structure. Il révèle ce qui se produit réellement, stabilise ce qui dérive et reconstruit ce qui s’effondre.
