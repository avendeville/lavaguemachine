Architecture de la Vague Machine

← Retour à l’accueil

L'architecture de la Vague Machine organise les méthodes, les couches conceptuelles et les structures qui permettent de lire, stabiliser et réparer les institutions. Elle ne s’intéresse ni aux intentions ni aux comportements individuels : elle analyse les formes, les dynamiques et les structures institutionnelles.

Elle repose sur trois méthodes opératoires et deux couches conceptuelles transversales.

1. Lire les mécaniques

La première méthode identifie les mécaniques institutionnelles : les mouvements qui structurent les organisations. Elle révèle les forces, les tensions et les dérives qui traversent les systèmes institutionnels.

2. Stabiliser le langage

La deuxième méthode, la Communication positive institutionnelle, clarifie les catégories, stabilise les formes et neutralise les dérives. Elle réintroduit du réel dans les échanges institutionnels.

3. Réparer l’institution

La troisième méthode, le Pôle communication‑santé, restaure les équilibres institutionnels, régule les tensions et reconstruit les dynamiques de travail. Elle agit comme un organe de soin institutionnel.

Les couches conceptuelles

Deux couches transversales structurent l’ensemble de la Vague Machine :

  • Le lexique conceptuel — les formes, catégories et notions opératoires.
  • Les architectures institutionnelles — les structures qui organisent les systèmes institutionnels.

Cosmologie institutionnelle

La cosmologie situe les institutions dans un ensemble plus vaste : leurs forces, leurs cycles, leurs équilibres et leurs effondrements. Elle constitue la vision d’ensemble de la Vague Machine.
