Organiser les méthodes. Structurer les couches conceptuelles. Lire les formes institutionnelles.
L’architecture de la Vague Machine décrit la structure qui permet de lire, stabiliser et réparer les institutions.

Elle ne s’intéresse ni aux intentions ni aux comportements individuels : elle analyse les formes, les dynamiques et les structures institutionnelles.

Les trois méthodes opératoires

1. Lire les mécaniques
Identifier les mécaniques institutionnelles : les mouvements qui structurent les organisations.
Lire les forces, les tensions et les dérives qui traversent les systèmes.

2. Stabiliser le langage
La méthode de Communication positive institutionnelle.
Clarifier les catégories, stabiliser les formes, neutraliser les dérives.
Réintroduire du réel dans les échanges institutionnels.

3. Réparer l’institution
Le Pôle communication‑santé.
Restaurer les équilibres, réguler les tensions, reconstruire les dynamiques de travail.
Un organe de soin institutionnel.

Les couches conceptuelles

Deux couches transversales structurent l’ensemble du système :

Le lexique conceptuel
Les formes, catégories et notions opératoires.

Les architectures institutionnelles
Les structures qui organisent les systèmes institutionnels.

Cosmologie institutionnelle

La cosmologie situe les institutions dans un ensemble plus vaste : leurs forces, leurs cycles, leurs équilibres et leurs effondrements.
Elle constitue la vision d’ensemble de la Vague Machine.
